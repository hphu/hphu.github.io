Pyview

Pyview is a data visualization tool that allows users to see their python program's call stack. Circles represent function calls and the nested circles make it easy to see what each function inside of it. The size of circles are correlated to runtime of a function allowing users to find bottlenecks or compare the efficiency of different subsystems of a program. Created in collaboration with Ryan Collins and Alex Doliner, Pyview was made in 24 hours for the 2013 Yale Hackathon.


Tender

Play as a schoolboy who journeys through two different realities. He must fight monsters in a stagnant yet colorful world as well as survive a mundane school life. Game mechanics change based on narrative. Coded using actionscript3 in flash, art assets drawn with Adobe PS5.

Tumblr Word Cloud

Scrapes up to a hundred tumblr user's recent posts and generates a word cloud with D3 using word occurences as size. The word cloud is then uploaded to imgur and able to be reblogged easily.


Tic Tac Toe Web Application

A multiplayer Tic Tac Toe game created for CS411 (Software Engineering). Written in HTML/CSSm, Javascript with PHP and MySQL backend following MVC architecture. Features include random matchmaking with another player online, Playing against Computer, Challenging/Accepting Friend games and Achievements.

Simon Says

Simon is a classic memory game where the player memorizes increasingly long combinations of colored lights and attempts to replicate them. I created the game for the Android application platform in Java. While the gameplay stays true to Simon, sleek graphics and customizable settings allow for a much more modern experience.
