Pyview

Pyview is a data visualization tool that allows users to see their python program's call stack. Circles represent function calls and the nested circles make it easy to see what each function inside of it. The size of circles are correlated to runtime of a function allowing users to find bottlenecks or compare the efficiency of different subsystems of a program. Created in collaboration with Ryan Collins and Alex Doliner, Pyview was made in 24 hours for the 2013 Yale Hackathon.
